### Manual Deployment

If you prefer to deploy manually:

```bash
# Clone the repository
git clone https://github.com/20csm2k25-netizen/cloud-portfolio.git
cd cloud-portfolio

# Checkout the improved UI branch
git checkout copilot/improve-portfolio-ui

# Merge to main
git checkout main
git merge copilot/improve-portfolio-ui --allow-unrelated-histories

# Push to GitHub
git push origin main
```

### Verify GitHub Pages Settings
